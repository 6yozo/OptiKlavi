Corpora is expected to be here.

The corpora files must be in UTF-8 encoding and contain sentences. One sentence per line.

Corpora can be downloaded from sources like: 


British National Corpus
http://www.natcorp.ox.ac.uk/

Wikipedia corpora for Hungarian and English
https://wortschatz.uni-leipzig.de/en/download/Hungarian

Hungarian web corpus
http://mokk.bme.hu/resources/webcorpus/
